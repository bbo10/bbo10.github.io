Project 1: 독일 은행 신용 리스크 예측 모델 구축

설명: 머신러닝을 활용해 채무 불이행 고객을 선제적으로 식별하고, 비즈니스 목표에 부합하는 객관적인 대출 심사 가이드라인을 구축한 프로젝트

핵심성과:

  • 모델 성능 향상: 모델의 전체 정확도를 57.9%에서 65.6%로 약 8%p 향상
  • 최적 모델 선정: 비즈니스 목표(부실 고객 식별)에 맞춰 평가지표를 설정, XGBoost 모델(F1-Score: 0.40)을 최종 채택
  • 전략 제안: 모델 예측 결과를 바탕으로 '승인/조건부 승인/거절' 3단계 대출 심사 가이드라인 제시

사용 기술:

Python Pandas & Numpy Scikit-learn Matplotlib & Seaborn

Project 2: Instacart 고객 재구매율 증대를 위한 맞춤형 마케팅 전략 수립

설명: SQL과 RFM 분석을 통해 100만 건 이상의 복잡한 구매 데이터를 분석하여 핵심 고객 그룹을 정의하고, 이들의 재구매율과 고객 가치를 극대화하는 실행 가능한 마케팅 전략을 제안한 프로젝트

핵심성과:

  • 고객 세분화: 전체 고객을 '우수(0.7%)', '잠재(38.0%)', '관심 필요(54.1%)', '이탈 위험(7.2%)' 4개의 핵심 그룹으로 분류하고, 성장의 열쇠가 '관심 필요 고객'에게 있음을 발견.
  • 전략적 연결고리 발견: 고객들의 주요 구매 패턴('신선식품 소량 반복 구매')과 핵심 고객 그룹('관심 필요 고객')을 연결하여, '신선식품 정기배송 구독' 모델이라는 구체적이고 설득력 있는 솔루션 도출.
  • 데이터 기반 전략 수립: 각 고객 그룹의 특성에 맞는 차별화된 액션 플랜(VIP 혜택, 웰컴백 리워드 등)을 설계하여 한정된 마케팅 리소스의 효율적 분배 방안 제시.

사용 기술:

Python SQL RFM Matplotlib

Project 3: 원석 액세서리 쇼핑몰 '선택과 집중' 성장 전략

설명: 지속적인 매출 하락을 겪고 있는 쇼핑몰의 광고, 주문, 리뷰 데이터를 종합 분석하여 문제의 근본 원인을 진단하고, 한정된 자원을 '될 놈'(고객과 상품)에 집중하여 성장 모멘텀을 회복시키는 데이터 기반의 '선택과 집중' 전략을 제안한 프로젝트

핵심 성과:

  • 문제 원인 진단: ROAS 분석을 통해 광고 효율이 과거 대비 29% 감소했음을, 코호트 분석을 통해 재구매가 첫 3개월 이내에만 집중되는 낮은 리텐션 문제를 정량적으로 규명.
  • '될 놈' 상품 발굴: 매출 상위 12개 상품에 대해 단계적 필터링(Step-wise Filtering) 기법을 적용, '단기 인기', '회원 기반 약함', '단종' 상품을 제외하고 잠재력 높은 新 스테디셀러 후보 4개를 발굴.
  • 매출 증대 시뮬레이션: 전체 고객의 69%를 차지하는 '비회원'의 첫 구매 상품을 '회원 인기 상품'으로 전환할 경우, 총매출이 16% 상승할 수 있음을 데이터 시뮬레이션으로 입증.
  • 제품 개선점 도출: LLM(Gemini) API를 활용하여 3천여 건의 고객 리뷰를 분석, '줄이 얇다', '사이즈가 작다' 등 구체적인 제품 개선점을 도출하여 상품 기획에 반영할 근거 마련.

사용 기술:

Python Tableau Cohort Analysis LLM API (Gemini)
